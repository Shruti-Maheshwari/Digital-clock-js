# Digital-clock-js
"A sleek responsive digital clock built with Javascript ,HTML, and CSS featuring real time updates.
3
**🚀 Features**
Real-time Updates: Displays hours, minutes, and seconds accurately, updating every second.
Modern UI: Minimalist design with a focus on readability and style.
Format: Displays time in a clear digital format (HH:MM:SS) with AM/PM modification.


**🛠️ Built With**
HTML5: Structure of the clock.
CSS3: Custom styling, layout, and responsiveness.
JavaScript (ES6): Core logic using the Date() object to fetch and display the time.
